## 📌 Overview
This project is part of my IoT Internship at CodeAlpha.

## 💡 Project Title
Motion Detection Alarm System using PIR Sensor

## 🧠 Description
This project demonstrates a motion detection system using a PIR sensor and Arduino. The PIR sensor detects human movement and sends a signal to the Arduino.

When motion is detected, an LED and buzzer are activated to alert the user. When no motion is detected, both remain OFF.

## 🔧 Components Used
- Arduino Uno
- PIR Motion Sensor
- LED
- Buzzer
- 220Ω Resistor

## ⚙️ Working Principle
The PIR sensor detects infrared radiation changes caused by human movement. When motion is detected, it outputs a HIGH signal to the Arduino (Pin 2). The Arduino then activates the LED and buzzer.
